# IntDorSys-Landing

Лендинг для [IntDorSys](https://github.com/Ourobor0s3/IntDorSys) — системы управления прачечной в общежитии. Содержит интерактивный превью Telegram-бота, полностью соответствующий поведению реального бота.

## Стек

- **Vue 3** (Composition API, `<script setup>`) + **Vite 6**
- **Тёмная/светлая тема** — сохраняется в `localStorage`
- **RU/EN локализация** — сохраняется, соответствует стилю Angular-приложения

## Команды

```bash
npm install
npm run dev     # → http://localhost:5173
npm run build   # → dist/
npm run preview # предпросмотр production-сборки
```

## Структура проекта

```
src/
├── main.ts               # инициализация, тема и локаль из localStorage
├── App.vue               # корень — NavBar, секции, Footer, переключатели
├── style.css             # CSS-переменные, сброс, утилиты (btn, card, .section)
├── i18n.ts               # RU/EN переводы, useI18n() composable
└── components/
    ├── NavBar.vue            # липкая навигация с якорями
    ├── HeroSection.vue       # герой + одна кнопка (ведёт в бэкенд)
    ├── FeaturesSection.vue   # 6 карточек возможностей
    ├── HowItWorksSection.vue # 3 шага регистрации
    ├── BotPreviewSection.vue # интерактивный макет телефона с inline-клавиатурой
    ├── TechStackSection.vue  # 8 карточек технологий
    ├── FooterSection.vue     # копирайт
    ├── LangToggle.vue        # RU / EN
    └── ThemeToggle.vue       # светлая / тёмная тема
```

## Тема

Переключатель в правом верхнем углу. Светлая/тёмная тема управляется через атрибут `data-bs-theme` на `<html>`, CSS-переменные в `style.css`.

## Локализация

Переключатель рядом с темой. По умолчанию RU. Сообщения бота внутри макета телефона всегда на русском (реальный Telegram-бот не имеет английской версии).

## Превью бота

`BotPreviewSection.vue` — реалистичный макет телефона:

- **Вкладка «Пользователь»** — полный поток регистрации: `/start` → ответ с ФИО → ожидание → подтверждение → меню
- **Вкладка «Администратор»** — интерактивная inline-клавиатура: клик по кнопкам переключает даты, записи, списки пользователей. Повторяет паттерн `EditMessageTextAsync` реального бота.

## Сборка

Результат — статический SPA в `dist/`. Роутера нет — одна страница, навигация по якорям. На сервере Nginx раздаёт статику и проксирует `/api/` на .NET-бэкенд.

```bash
npm run build
```
